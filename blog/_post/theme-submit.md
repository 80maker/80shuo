## 欢迎在留言区提交主题

### 格式说明

```
---
name: vuepress-theme-maker // 主题包名称
cover: https://80shuo.com/images/screenshot.webp // 截图
author: 80maker // 可选 默认从npm包信息获取
email: cmgddd@126.com // 用于头像图片生成
site: https://80shuo.com // 主题预览地址
theme-tag:
  - dark
  - blog
---
```
